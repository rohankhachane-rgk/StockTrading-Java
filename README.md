# Stock Trading Simulator

## Description

Stock Trading Simulator is a Java-based application that simulates stock market trading. Users can register, manage portfolios, buy and sell stocks, and monitor investment performance using simulated market data.

## Features

* User registration and login
* Simulated stock market
* Portfolio management
* Buy and sell stocks
* Investment performance tracking
* Real-time price updates

## Technologies Used

* Java
* Collections Framework
* Timer and TimerTask
* Object-Oriented Programming (OOP)

## How to Run

1. Compile the program:
   javac StockTrading.java

2. Run the program:
   java StockTrading
